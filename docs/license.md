::: tip 开始之前请先看一下
本页是几条**轻量级的小提醒**，并不是正式协议或合同。
开始使用之前希望你简单读一下，理解后再继续。
:::

本扩展是粉丝个人制作的非官方小工具，与 Cygames 及《碧蓝幻想》（GBF）
运营方**没有任何关系**，也未获得其认可或合作。

## 唯一一条希望你配合的事

::: warning 请尽量不要把本扩展放进直播 / 视频里
- 请尽量**不要**在直播（含实时直播、录播、视频投稿、剪辑等）中使用本扩展。
- 也希望本扩展的画面**不要出现**在直播 / 视频画面中。
:::

理由很简单：直播 / 视频的观众里会混入不熟悉浏览器扩展或对效率化工具
持负面看法的人。被这类目光看到之后，整个《碧蓝幻想》玩家社区里
使用浏览器扩展的环境都会变得更紧张，最终所有使用者都会变得不好过。
为了避免这种情况，希望大家在直播 / 视频之外的场合、低调地个人使用。

除此之外没有别的硬性要求 —— SNS 上的分享、向身边朋友推荐、
个人范围内的自由使用等，都不会特别限制。

## 关于现在的开发 / 测试阶段

本扩展现在仍处于**开发 / 测试中**的阶段，账号封停（BAN）等风险
**还不太明确**。这段开发 / 测试期间预计**持续到 2027 年 5 月左右**，
在那之前希望大家把使用规模和分享范围都控制在小范围内、低调地用。

本扩展本身**不包含**宏、BOT、加速、自动化等任何违规功能，只是
出于谨慎，请在这段观察期内配合上述做法。2027 年 5 月之后会根据
实际情况再考虑是否放宽。
