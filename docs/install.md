# 安装方法

本扩展以**未打包扩展（开发者模式）**的形式安装。

::: warning 安装即代表同意条款
开始安装与使用前，请先阅读并同意 [许可协议 / 使用条款](/license)。
:::

## 步骤

1. 取得本扩展的文件夹（包含 `manifest.json` 的那个文件夹）。
2. 打开 Chrome，在地址栏输入并访问：

   ```
   chrome://extensions
   ```

3. 打开右上角的 **「开发者模式」** 开关。
4. 点击 **「加载已解压的扩展程序」**。
5. 选择第 1 步中那个**包含 `manifest.json` 的文件夹**。
6. 扩展安装完成后，建议点击工具栏的拼图图标，将 HamuBle **固定**到工具栏。

## 打开侧边栏

1. 打开《碧蓝幻想》网页版页面。
2. 点击工具栏上的 HamuBle 图标，打开侧边栏。
3. 在侧边栏中即可看到救援列表与过滤选项。

## 更新

当获得新版本的文件夹后：

1. 用新版本文件替换原文件夹内容。
2. 回到 `chrome://extensions`。
3. 点击 HamuBle 卡片上的**刷新 / 重新加载**按钮。

## 卸载

在 `chrome://extensions` 页面找到 HamuBle，点击**移除**即可。
