# 使用方法

![侧边栏整体外观](/img/usage/sidepanel-overview.jpg)

打开《碧蓝幻想》页面后，点击工具栏的 HamuBle 图标打开侧边栏。
侧边栏顶部依次为 **仪表盘 / 救援筛选 / 我的任务 / 信息 / 设置** 五个入口，
此外还有一条 **收藏** 栏，其底端是 **主页注册**（房子）按钮。本页按这些入口分别详细说明。

::: tip 提示
本扩展只整理与显示信息，**不会自动操作游戏**。所有最终操作（参战、开始、点击等）
仍由你自己手动完成。
:::

## 主页注册

收藏栏底端的主页（房子）图标（与收藏槽位以分隔线区分），用于把当前游戏页面快速设为浏览器主页按钮。

### 功能与操作

![点击主页图标后复制网址并打开 Chrome 设置页](/img/usage/home-register.gif)

点击该图标后，扩展会：

1. **复制当前《碧蓝幻想》页面的网址**到剪贴板；
2. 自动打开 Chrome 的设置页面（外观 / 主页按钮）。

之后只需在设置页把网址**手动粘贴并保存**，即可用浏览器主页按钮一键回到该页面。

### 注意事项

- 扩展只复制网址并打开设置页，**粘贴与保存仍由你手动完成**，不会自动更改浏览器设置。
- 若未检测到《碧蓝幻想》标签页，按钮会变红提示，请先打开游戏页面再点击。

## 救援筛选

「救援筛选」是主功能。侧边栏会整理显示当前的救援多人房间列表，
当游戏页面的救援列表变化时，侧边栏会随之自动更新。
可用下列条件自由组合进行筛选与排序。

### 救援列表

![救援房间卡片示例](/img/usage/raid-list-card.jpg)

列表中每个房间以卡片显示，包含 Boss 缩略图、任务名、当前 HP、参战人数、
剩余时间与房主等信息。点击卡片即可前往对应的救援页面（最终参战仍由你手动确认）。

### HP 过滤

![滑动 HP 阈值与切换以上/以下时列表实时筛选](/img/usage/hp-filter.gif)

按 Boss 当前 HP 的百分比筛选房间。

- **数值范围**：20% ~ 100%，**滑块（步进 10%）** 调节。默认值 50%。下限设为 20% 是因为 0%~10% 实际不会有可参战的房间，过滤价值低。
- **判定方向**：通过「以上 / 以下」**切换按钮**选择。激活的一侧以高亮色显示，未激活的一侧呈灰色。
  - **以上**：仅显示 HP **≥ 设定值** 的房间。用于挑选刚开（高 HP）的房间。
  - **以下**（默认）：仅显示 HP **≤ 设定值** 的房间。用于挑选接近击破的房间。
- 滑块右侧实时显示当前阈值，操作时立即生效，无需保存按钮。
- 关闭复选框时整个 HP 过滤块呈淡化状态，不参与筛选。

### 参战人数过滤

![调整人数阈值与切换以上/以下](/img/usage/member-filter.gif)

按当前已参战的人数筛选房间。

- **数值范围**：1 ~ 6 人，**滑块（步进 1）** 调节。默认值 5 人。
- **判定方向**：通过「以上 / 以下」**切换按钮**选择。激活的一侧以高亮色显示，未激活的一侧呈灰色。
  - **以下**（默认）：仅显示参战人数 **≤ 设定值** 的房间。
    - 例：`1 人以下` 用于抢「全新房间」（无人参战或仅自发）。
  - **以上**：仅显示参战人数 **≥ 设定值** 的房间。
    - 例：`2 人以上` 用于「蹭血」（已有同伴削过 HP 的房间）。
- 上限设为 6 是为了覆盖人数上限 6 的多人副本「满员」状态。再高的设定（7 人以上）多数对应「已满员无法参战」或「Boss 已被击破」的状态，过滤价值低。
- 旧版本保存了 7 以上数值的用户，启动时会自动夹紧到 6。HP 同样会从 20% 以下夹紧到 20%。已保存的过滤模板恢复时同样夹紧。

### FP 对象限定

![仅显示消耗 FP 的房间](/img/usage/fp-filter.jpg)

只显示会消耗 FP 的房间，方便集中处理与 FP 相关的目标。

### 排序

![切换排序方式](/img/usage/sort.gif)

提供多种排序方式：默认顺序、按 HP、按参战人数、按剩余时间等，
可随时切换，让你想要的房间更靠前。

### 过滤模板

![保存、切换与删除过滤模板](/img/usage/template-save.gif)

把常用的过滤组合（HP / 人数 / FP 等条件）保存为**模板**：

- 输入名称即可保存当前的筛选条件；
- 之后在不同场景间**一键切换**，不必每次手动重设；
- 不需要的模板可以删除。

### 自动隐藏已耗尽

可隐藏已经无法参加 / 已耗尽的房间，让列表更清爽。
该开关也可在「设置」中统一调整。

### 已参战的救援（绿框显示 / 隐藏）

已参战的救援会以**绿色边框**标示，方便与未参战的房间区分。判定为「已参战」的情形有两种：

- 房间在游戏内带有**参战中图标**（包括自己发现 / 自发的多人战）；
- 在本扩展列表中**点击参战**过的房间（列表刷新前一直保持绿框；游戏内刷新列表后改由参战中图标判定）。

如果不想在列表中看到已参战的房间，可在「设置」中开启「隐藏已参战的救援」。
开启后，点击参战的一瞬间该卡片就会从列表中移除，顶部的房间数也会随之减少。

### 列表信息与 BP 锁定

![BP 不足时列表临时锁定](/img/usage/bp-lock.jpg)

- 列表上方会显示当前**筛选后的房间数**、**当前 BP** 与**最近更新时间**；
- 当 BP 不足以参加时，列表会**临时锁定**并给出提示；BP 恢复后会自动解除锁定。

## 我的任务

「我的任务」会记录你**自己开启（自发）过的任务**，方便日后再次开启。

### 任务磁贴的信息

![任务磁贴示例](/img/usage/task-tile.jpg)

每个任务以磁贴显示，包含：缩略图、任务名、
**剩余可开启次数 / 当日已开启次数**、最近开启时间。
点击磁贴即可前往对应的再自发页面（最终开启仍由你手动确认）。

### 分类筛选

![通过分类标签筛选任务](/img/usage/task-category.gif)

通过顶部的分类标签筛选要显示的任务：

- 分类包括 活动 / PRO / NORMAL / HIGH LEVEL / UNLIMITED / 自由 / 其他 等；
- 「ALL」为单选（显示全部），其余分类可**多选**组合；
- 当某分类内任务全部完成时，会显示已达成标记。

### Hell 任务跳过次数

对于支持跳过的 Hell 任务，可在磁贴上选择**本次开启时跳过的次数**，
按需要调整后再前往开启。

### 完成的任务（已耗尽）

当某任务的可开启次数耗尽时，磁贴会变灰显示；
可在「设置」中开启「自动隐藏已耗尽」将其隐藏。

### 活动信息

![活动横幅（预告 / 举办中 / 已结束）](/img/usage/event-banner.jpg)

活动期间，列表上方会显示与活动相关的信息，并按
**预告 / 举办中 / 已结束**分组列出；每条可单独删除。
是否显示可在「设置」的「活动横幅」中开关。

### 每日刷新

当日开启次数会按游戏日（约每天 5:00）自动刷新统计。

### 自动清理

为避免列表被临时开启的任务堆满，部分记录会在一段时间未使用后自动移除：

- **其他**分类的任务：最近一次自发**超过 2 天**未再开启时，自动从列表移除。
- **活动**分类的任务：与活动结束同步移除（活动结束后约保留数日的缓冲）。
- 其余分类（PRO / NORMAL / HIGH LEVEL / UNLIMITED / 自由 等）**不会自动删除**，会一直保留，直到你手动清除。

## 掉落监视

「掉落监视」会在你**通关战斗的结算画面**自动识别掉落物，并把你登记过的素材按日期累计到**仪表盘的掉落日历**中，方便回顾「这个月刷到了几个目标素材」。

::: tip 提示
检测为**只读**：扩展只读取结算画面已经显示出来的掉落，不会自动点击、不会操作游戏，也不会调用任何游戏接口。
:::

扩展预置了 2 个常用素材作为示例（绯绯色金 ヒヒイロカネ、刻之流沙 刻の流砂），可直接使用，也可删除后换成自己需要的。

### 登记监视物品

有三种方式登记要监视的素材，最多可登记 **20 项**：

- **从「本场」页脚一键登记（推荐）**：打过一场战斗后，页脚会列出本场所有掉落的候选图标，点击想监视的图标即可登记；登记后图标右上角会出现 `✓`。
- **在设置里粘贴图标网址**：打开「设置 → 掉落监视」，把素材 / 武器的**图标网址**粘贴到输入框，点击「添加」（或按回车）。网址形如 `…/assets/物品分类/s/编号.jpg`，在游戏内右键素材图标 →「复制图片地址」即可获得。
- **把图标拖到输入框**：直接把游戏内的素材图标**拖拽**到设置的输入框 —— 网址会自动填入，省去手动复制，随后点击「添加」即可完成登记。

::: tip 提示
网址支持 `m` / `s` / `b` 各种尺寸与 `jpg` / `png`，扩展会自动归一化；物品分类支持单层（`weapon`、`summon` 等）与双层（`item/article`、`item/evolution` 等）。重复登记会被忽略，登记满 20 项后会拒绝继续添加。
:::

### 「本场」页脚（最近掉落）

侧边栏底部常驻一条「本场」页脚，显示**最近一场战斗**识别到的全部掉落（仅显示最新一场，不累计、也不写入存储）：

- 已登记的图标带 `✓` 标记；未登记的点击即可登记。
- 数量大于 1 的掉落会显示 `×N`。
- 还没打过战斗时，显示「打过一场战斗后这里会出现候选」。

### 掉落日历（仪表盘）

在**仪表盘**中，监视素材的累计掉落以**月历**形式显示：

- **周一开始**，周末以别色区分，今日高亮。
- 每个日期格内列出当天掉到的各监视素材图标与数量（`×N`）。
- 顶部 `‹` `›` 可翻看前后月份；翻到当月时，向后（下个月）按钮会自动禁用。
- 数据最多保留 **500 条 / 90 天**（约 3 个月），超出后自动清理较旧的记录。

### 删除与清空

- **删除单个监视物品**：在「设置 → 掉落监视」点击已登记的图标即可删除；该素材**已累计的掉落记录也会一并清除**，日历中随之消失。
- **清空全部记录**：「设置 → 掉落监视」右上角的「清空记录」按钮会清空**所有已记录的掉落事件**（日历清零），但**不会删除**你登记的监视物品，方便开新一期活动时从零开始统计。

## 设置

![设置面板展开后的样子](/img/usage/settings-panel.jpg)

点击侧边栏的齿轮图标展开设置面板，集中管理显示相关选项。各项说明如下。

### 图标栏位置

![切换左右位置时图标栏与收藏栏镜像变化](/img/usage/icon-bar-position.gif)

决定功能图标栏在侧边栏中的放置位置。

- `左`：图标栏置于侧边栏左侧（默认）；
- `右`：图标栏置于右侧，收藏栏也会一并镜像到右侧。

### 我的任务列数

「我的任务」磁贴网格的列数。数值越大，每行显示的磁贴越多、单个越小。

- 可选 `3` / `4` / `5` / `6`（默认 `3`）。

### 自动隐藏已耗尽

是否自动隐藏已耗尽 / 无法再开启的对象。

- `开`：自动隐藏，列表更清爽（默认）；
- `关`：仍然显示，已耗尽对象以变灰方式呈现。

### 活动横幅

是否在「我的任务」上方显示活动信息横幅。

- `开`：显示「预告 / 举办中 / 已结束」活动条目（默认）；
- `关`：不显示活动信息横幅。

### 隐藏已参战的救援

是否把已参战（带参战中图标，或在列表中点击参战过）的救援从救援列表中隐藏。

- `开`：隐藏已参战的房间；开启状态下点击参战后卡片会立即从列表移除，房间数随之减少；
- `关`：仍然显示，已参战的房间以绿色边框标示（默认）。

### 版本号

设置面板底部会显示当前扩展的版本号（仅供查看，不可更改）。

## 收藏

![登记、跳转与删除收藏书签的操作](/img/usage/favorites.gif)

侧边栏的收藏栏提供 **5 个网址书签位**，用于一键回到常用页面。

### 登记收藏

点击空位（显示 `+`）时，会把**当前标签页的网址**登记到该位，
按钮随即变为对应编号（`1`〜`5`）。

### 跳转

点击已登记的编号（`1`〜`5`），当前标签页会前往该收藏的网址。

### 删除

将鼠标移到已登记的按钮上会出现 `×`，点击即可删除该收藏，按钮恢复为 `+`。
