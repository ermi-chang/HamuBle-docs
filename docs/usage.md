# 使用方法

打开《碧蓝幻想》页面后，点击工具栏的 HamuBle 图标打开侧边栏。
侧边栏顶部依次为 **主页注册 / 救援筛选 / 我的任务 / 信息 / 设置** 五个入口，
此外还有一条 **收藏** 栏。本页按这些入口分别详细说明。

::: tip 提示
本扩展只整理与显示信息，**不会自动操作游戏**。所有最终操作（参战、开始、点击等）
仍由你自己手动完成。
:::

## 主页注册

侧边栏顶部的主页（房子）图标，用于把当前游戏页面快速设为浏览器主页按钮。

### 功能与操作

点击该图标后，扩展会：

1. **复制当前《碧蓝幻想》页面的网址**到剪贴板；
2. 自动打开 Chrome 的设置页面（外观 / 主页按钮）。

之后只需在设置页把网址**手动粘贴并保存**，即可用浏览器主页按钮一键回到该页面。

### 注意事项

- 扩展只复制网址并打开设置页，**粘贴与保存仍由你手动完成**，不会自动更改浏览器设置。
- 若未检测到《碧蓝幻想》标签页，按钮会变红提示，请先打开游戏页面再点击。

## 救援筛选

「救援筛选」是主功能。侧边栏会整理显示当前的救援多人房间列表，
当游戏页面的救援列表变化时，侧边栏会随之自动更新。
可用下列条件自由组合进行筛选与排序。

### 救援列表

列表中每个房间以卡片显示，包含 Boss 缩略图、任务名、当前 HP、参战人数、
剩余时间与房主等信息。点击卡片即可前往对应的救援页面（最终参战仍由你手动确认）。

### HP 过滤

按 Boss 当前 HP 的百分比筛选：

- 可选择只显示 HP 在某数值**以上**或**以下**的房间；
- 适合挑选「接近击破」或「刚开（满血）」的房间。

### 参战人数过滤

按当前参战人数筛选，例如只显示**缺人**（人数较少）的房间，便于补位救援。

### FP 对象限定

只显示会消耗 FP（应援点数）的房间，方便集中处理与 FP 相关的目标。

### 排序

提供多种排序方式：默认顺序、按 HP、按参战人数、按剩余时间等，
可随时切换，让你想要的房间更靠前。

### 过滤模板

把常用的过滤组合（HP / 人数 / FP 等条件）保存为**模板**：

- 输入名称即可保存当前的筛选条件；
- 之后在不同场景间**一键切换**，不必每次手动重设；
- 不需要的模板可以删除。

### 自动隐藏已耗尽

可隐藏已经无法参加 / 已耗尽的房间，让列表更清爽。
该开关也可在「设置」中统一调整。

### 列表信息与 FP 锁定

- 列表上方会显示当前**筛选后的房间数**、**当前 FP** 与**最近更新时间**；
- 当 FP 不足以参加时，列表会**临时锁定**并给出提示；FP 恢复后会自动解除锁定。

## 我的任务

「我的任务」会记录你**自己开启（自发）过的任务**，方便日后再次开启。

### 任务磁贴的信息

每个任务以磁贴显示，包含：缩略图、任务名、
**剩余可开启次数 / 当日已开启次数**、最近开启时间。
点击磁贴即可前往对应的再自发页面（最终开启仍由你手动确认）。

### 分类筛选

通过顶部的分类标签筛选要显示的任务：

- 分类包括 活动 / PRO / NORMAL / HIGH LEVEL / UNLIMITED / 自由 / 其他 等；
- 「ALL」为单选（显示全部），其余分类可**多选**组合；
- 当某分类内任务全部完成时，会显示已达成标记。

### Hell 任务跳过次数

对于支持跳过的 Hell 任务，可在磁贴上选择**本次开启时跳过的次数**，
按需要调整后再前往开启。

### 完成的任务（已耗尽）

当某任务的可开启次数耗尽时，磁贴会变灰显示；
可在「设置」中开启「自动隐藏已耗尽」将其隐藏。

### 活动信息

活动期间，列表上方会显示与活动相关的信息，并按
**预告 / 举办中 / 已结束**分组列出；每条可单独删除。
是否显示可在「设置」的「活动横幅」中开关。

### 每日刷新

当日开启次数会按游戏日（约每天 5:00）自动刷新统计。

## 信息

「信息」页提供使用相关的入口与支持信息。

### 在线使用手册

跳转到本在线文档（即你正在阅读的使用手册），查看完整说明与常见问题。

### 支持作者

提供作者支持渠道的链接；如果你喜欢本扩展，欢迎前往支持。

### 项目与版本

提供项目仓库（GitHub）链接，并显示当前扩展的**版本号**。

## 设置

点击侧边栏的齿轮图标展开设置面板，集中管理显示相关选项。各项说明如下。

### 图标栏位置

决定功能图标栏在侧边栏中的放置位置。

- `左`：图标栏置于侧边栏左侧（默认）；
- `右`：图标栏置于右侧，收藏栏也会一并镜像到右侧。

### 我的任务列数

「我的任务」磁贴网格的列数。数值越大，每行显示的磁贴越多、单个越小。

- 可选 `3` / `4` / `5` / `6`（默认 `3`）。

### 自动隐藏已耗尽

是否自动隐藏已耗尽 / 无法再开启的对象。

- `开`：自动隐藏，列表更清爽（默认）；
- `关`：仍然显示，已耗尽对象以变灰方式呈现。

### 活动横幅

是否在「我的任务」上方显示活动信息横幅。

- `开`：显示「预告 / 举办中 / 已结束」活动条目（默认）；
- `关`：不显示活动信息横幅。

### 版本号

设置面板底部会显示当前扩展的版本号（仅供查看，不可更改）。

## 收藏

侧边栏的收藏栏提供 **5 个网址书签位**，用于一键回到常用页面。

### 登记收藏

点击空位（显示 `+`）时，会把**当前标签页的网址**登记到该位，
按钮随即变为对应编号（`1`〜`5`）。

### 跳转

点击已登记的编号（`1`〜`5`），当前标签页会前往该收藏的网址。

### 删除

将鼠标移到已登记的按钮上会出现 `×`，点击即可删除该收藏，按钮恢复为 `+`。
