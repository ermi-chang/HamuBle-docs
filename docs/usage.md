# 使用方法

打开《碧蓝幻想》页面后，点击工具栏的 HamuBle 图标打开侧边栏，
即可使用以下功能。

::: tip 提示
本扩展只整理与显示信息，**不会自动操作游戏**。所有最终操作仍由你手动完成。
:::

## 救援列表

侧边栏会整理显示当前的救援多人房间列表。当游戏页面的救援列表
发生变化时，侧边栏会随之更新。

## 过滤

### HP 过滤

可以按 Boss 的 HP 进行过滤，例如只显示 HP 在某个百分比**以上**
或**以下**的房间，方便挑选接近击破或刚开的房间。

### 参战人数过滤

可以按当前参战人数过滤，例如只显示人数较少（缺人）的房间。

### 自动隐藏已耗尽

可开启自动隐藏已经无法参加 / 已耗尽的房间，让列表更清爽。

## 排序

提供多种排序方式（如默认顺序、按 HP 等），可根据需要切换，
让你想要的房间更靠前。

## 过滤模板

把常用的过滤组合保存为**模板**，在不同场景间一键切换，
不必每次手动重设各项条件。

## 收藏

可以把关注的对象加入收藏，便于快速识别与查找。

## 活动信息显示

在活动期间，侧边栏可显示与当前活动相关的信息提示，
可根据需要开关。

## 显示设置

侧边栏提供若干显示相关的设置（如图标栏位置、历史显示列数等），
可按个人习惯调整。
