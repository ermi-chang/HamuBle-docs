# 使用方法

打开《碧蓝幻想》页面后，点击工具栏的 HamuBle 图标打开侧边栏。
侧边栏顶部的图标依次为 **主页注册 / 救援筛选 / 我的任务 / 信息 / 设置**，
另有一条 **收藏** 栏。下面按这些入口分别说明。

::: tip 提示
本扩展只整理与显示信息，**不会自动操作游戏**。所有最终操作仍由你手动完成。
:::

## 主页注册

点击侧边栏顶部的主页（房子）图标，会**复制当前《碧蓝幻想》页面的网址**，
并打开 Chrome 的设置页面，方便你把它设为浏览器的主页按钮，
之后点主页按钮即可快速回到该页面。

- 仅复制网址并打开设置页，**粘贴与保存仍由你手动完成**。
- 未检测到《碧蓝幻想》标签页时，按钮会变红提示，请先打开游戏页面。

## 救援筛选

侧边栏会整理显示当前的救援多人房间列表，游戏页面的救援列表变化时会随之更新。
可用以下条件组合筛选与排序：

- **HP 过滤**：按 Boss 的 HP 百分比，只显示某数值**以上**或**以下**的房间，
  便于挑选接近击破或刚开的房间。
- **参战人数过滤**：按当前参战人数筛选，例如只显示缺人的房间。
- **FP 对象限定**：只显示会消耗 FP 的房间。
- **排序**：提供默认顺序、按 HP、按人数、按剩余时间等多种排序方式。
- **过滤模板**：把常用的过滤组合保存为模板，在不同场景间一键切换。
- **自动隐藏已耗尽**：隐藏已无法参加 / 已耗尽的房间（可在「设置」中开关）。

列表上方会显示筛选后的房间数、当前 FP 与最近更新时间；
当 FP 不足以参加时，列表会临时锁定并给出提示，FP 恢复后自动解除。

## 我的任务

记录你**自己开启（自发）过的任务**，方便再次开启。每个任务以磁贴显示，
包含缩略图、任务名、剩余可开启次数 / 当日次数与最近开启时间，
点击磁贴即可前往对应的再自发页面。

- **分类筛选**：通过顶部分类标签（活动 / PRO / NORMAL / 等）筛选，
  「ALL」为单选、其余可多选；某分类全部完成会显示已达成标记。
- **Hell 任务跳过次数**：对可跳过的 Hell 任务，可在磁贴上选择本次开启的次数。
- **完成的任务**：次数耗尽的任务会变灰，可在「设置」中自动隐藏。
- **活动信息**：活动期间会在列表上方分「预告 / 举办中 / 已结束」显示活动条目，
  可逐条删除，是否显示可在「设置」中开关。

每日会按游戏日（约 5:00）刷新当日次数统计。

## 信息

提供使用相关的入口与支持信息：

- **使用手册**：跳转到本在线文档（即本页所在的手册）。
- **支持作者 / GitHub**：作者支持与项目仓库链接。
- **版本号**：显示当前扩展版本。

## 设置

点击侧边栏的齿轮图标展开设置面板，集中管理显示选项。各项说明如下：

- **图标栏位置**：决定功能图标栏在侧边栏中的放置位置。
  - `左`：图标栏置于左侧（默认）。
  - `右`：图标栏置于右侧，收藏栏一并镜像到右侧。
- **我的任务列数**：「我的任务」磁贴网格的列数；数值越大，每行磁贴越多、单个越小。
  - 可选 `3` / `4` / `5` / `6`（默认 `3`）。
- **自动隐藏已耗尽**：是否隐藏已耗尽 / 无法再开启的任务。
  - `开`：自动隐藏，列表更清爽（默认）。
  - `关`：仍显示，已耗尽任务以变灰方式呈现。
- **活动横幅**：是否在「我的任务」上方显示活动信息。
  - `开`：显示「预告 / 举办中 / 已结束」活动条目（默认）。
  - `关`：不显示活动信息横幅。
- **版本号**：面板底部显示当前扩展版本号（仅显示，不可更改）。

## 收藏

侧边栏的收藏栏提供 **5 个网址书签位**，用于一键回到常用页面：

- 点击空位（显示 `+`）会把**当前标签页的网址**登记到该位。
- 点击已登记的编号（`1`〜`5`）即可前往该网址。
- 鼠标移到按钮上会出现 `×`，点击可删除该收藏。
