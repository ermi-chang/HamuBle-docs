**不可以。** 禁止主播 / 直播者在直播中使用，禁止在 SNS 等平台
传播本扩展。这并非单纯的个人偏好，而是为了保护**全体使用者**

## 这个扩展会自动帮我打游戏吗？

不会。本扩展是**只读**的信息整理工具，不会自动点击、不会自动
战斗、不会自动加入房间。所有最终操作都需要你手动进行。

## 安装后侧边栏没有显示 / 没反应？

请依次确认：

1. 当前页面是否为《碧蓝幻想》网页版页面。
2. 是否在 `chrome://extensions` 中正常启用了本扩展。
3. 修改文件后，是否在 `chrome://extensions` 点击了**重新加载**。
4. 尝试刷新游戏页面后，再点击工具栏图标打开侧边栏。

## 列表没有更新？

侧边栏会跟随游戏页面的救援列表变化而更新。若长时间未更新，
可尝试刷新游戏页面，或重新打开侧边栏。

## 使用本扩展会被封号吗？

本扩展按只读原则设计，但它仍属于第三方扩展，**作者不对任何
后果（包括账号封停）负责**。请务必阅读 [使用条款](/license) 与本页
[免责声明](#disclaimer)，并在理解风险后自行决定是否使用。
低调、克制地个人使用，并遵守不直播、不传播的约定。

## 支持哪个浏览器？

Google Chrome 116 及以上版本。
