# 更新日志

## v1.3.0 (26-06-01)

- 事件显示改为横幅图片：开始前／进行中／已结束都以横幅展示，点击横幅可跳转到对应事件页面，× 可单独关闭
- 横幅固定为首次获取的那张图片，即使我的页面随机切换也不变；约两周无更新会自动清理
- 已结束的事件横幅会变灰并移到「已结束」
- 修复事件预告开始后同一事件重复显示两条的问题
- 预告刚升格、跳转地址尚未确定时横幅暂不可点击，打开一次正式事件页面后即可正常点击
