# 更新日志

## v1.4.0 (26-06-05)

- 仪表盘掉落历史：从 10 件列表改为月单位日历（周一开始，周末别色，前后月可翻）
- 数据保留期：从最近 10 件扩展到 500 件 / 90 天，过去 3 个月可回顾
- 战斗结算检测：修复 hell skip 等 skip 变体的掉落漏检
- 监视上限：最多 20 项（满了会拒绝继续添加）
- 删除监视物品：会一并清除已记录的事件
- 结算扫描：改为即时检测，秒退结算的高速周回也能记录到掉落
- 仪表盘版块顺序：调整为 自发 → 活动 → 掉落
- 掉落记录行：去掉名称列，日期精简为 MM-DD，名称改走悬停提示
- 图标尺寸：累计 30px、监视 chip 48px、页脚 chip 42px（整体放大约 1.5 倍）
- 掉落监视设置：去掉 [+] 折叠，URL 输入框常驻显示
- 页脚「本场」图标：改用 s/.jpg 与已注册项的本地缓存，二次显示更快
- 掉落图标：统一补 lazy load / async decode
