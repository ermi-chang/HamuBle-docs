# 关于 HamuBle

HamuBle 是一个用于《碧蓝幻想》（GBF）**救援多人**房间的
过滤与显示 Chrome 扩展。它在浏览器侧边栏中整理显示救援房间列表，
并提供 HP、参战人数、排序等过滤功能，帮助你更快找到想要参加的房间。

::: warning 在阅读本页之前
请先阅读 [许可协议 / 使用条款](/license)。本扩展为非官方工具，
**禁止在直播中使用、禁止在 SNS 上传播**。
:::

## 主要特性

- **救援列表过滤**：按 HP、参战人数等条件筛选救援房间。
- **排序**：提供多种排序方式，便于浏览。
- **自动隐藏**：可自动隐藏已耗尽 / 不需要的房间。
- **过滤模板**：保存常用的过滤组合，一键切换。
- **侧边栏显示**：在浏览器侧边栏中整理显示，浏览更轻松。
- **活动信息显示**：可显示当前活动相关信息。

## 设计原则

本扩展坚持**只读**设计：

- 仅**读取**游戏页面信息进行整理显示，**不会自动操作游戏**。
- **不会代替你点击、不会自动战斗、不会自动加入房间。**
- 页面跳转等操作均通过浏览器标准方式进行。

请始终把它当作一个“信息浏览与整理”工具，而非自动化工具来使用。

## 运行环境

- Google Chrome 116 及以上版本
- 仅在《碧蓝幻想》网页版页面上运行
